Neural voice cloning

The project focuses on developing a system that trains itself on a user's voice. It then uses the trained model to generate voice notes identical to the user's voice for textual input provided to it






















Features



Text to speech model

The model provides an audio for the given text on which it is trained. Right now it is trained on only one voice


Get started

Voice Conversion

The model converts a given audio input into a target voice. Right now it is trained on only one voice


Get started

Audio Book

This model converts any pdf into an audio book of custom voice.
